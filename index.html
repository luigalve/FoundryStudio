SHORTCUTS

Single keys, active only while you are not typing in a field. Click a key box and press a new key to rebind. Saved inside your project file. Arrow keys nudge the selection 1px (Shift = 10px); Ctrl+Z / Ctrl+Y are undo / redo; these are fixed.

Drag on empty canvas to marquee-select. Shift-click adds to selection. Hold Alt and click to cycle to the shape underneath. Each capsule downloads from its own header. Anim: off shows the still picture a renderer sees when it ignores motion; that picture has to look finished.
